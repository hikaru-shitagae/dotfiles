## 基本概念

WezTermには3つの階層があります：

1. **Workspace（ワークスペース）**: プロジェクトごとの作業環境
2. **Tab（タブ）**: ワークスペース内の複数のターミナル
3. **Pane（ペイン）**: タブ内で画面を分割したターミナル

```
Workspace
 └── Tab 1
 │    ├── Pane 1
 │    └── Pane 2
 └── Tab 2
      └── Pane 1
```

---

## Leader キー

**Leader キー: `Ctrl+q`**

多くの操作は Leader キーを押してから、別のキーを押すことで実行します。
例: `Ctrl+q` を押してから `d` を押すと、画面が縦に分割されます。

---

## Workspace（ワークスペース）

プロジェクトごとに独立した作業環境を作成できます。

| キー | 説明 |
|------|------|
| `Ctrl+q w` | ワークスペース一覧を表示して切り替え |
| `Ctrl+q Shift+W` | 新しいワークスペースを作成 |
| `Ctrl+q $` | 現在のワークスペース名を変更 |

**使い方の例**:
- プロジェクトAとプロジェクトBで別々のワークスペースを作成
- それぞれのワークスペースで独立したタブとペインを管理

## Tab（タブ）

### タブの作成と削除

| キー | 説明 |
|------|------|
| `Cmd+t` | 新しいタブを作成 |
| `Cmd+w` | 現在のタブを閉じる（確認あり） |

### タブの移動

| キー | 説明 |
|------|------|
| `Ctrl+Tab` | 次のタブへ移動 |
| `Ctrl+Shift+Tab` | 前のタブへ移動 |
| `Cmd+1` 〜 `Cmd+9` | 指定番号のタブへ直接移動（Cmd+9は最後のタブ） |

### タブの順序変更

| キー | 説明 |
|------|------|
| `Ctrl+q {` | 現在のタブを左に移動 |
| `Ctrl+q }` | 現在のタブを右に移動 |

---

## Pane（ペイン）

### ペインの作成と削除

| キー | 説明 |
|------|------|
| `Ctrl+q d` | 画面を縦に分割（下に新しいペイン） |
| `Ctrl+q r` | 画面を横に分割（右に新しいペイン） |
| `Ctrl+q x` | 現在のペインを閉じる（確認あり） |

**覚え方**:
- **d** = down（下）
- **r** = right（右）

### ペイン間の移動

| キー | 説明 |
|------|------|
| `Ctrl+q h` | 左のペインへ移動 |
| `Ctrl+q j` | 下のペインへ移動 |
| `Ctrl+q k` | 上のペインへ移動 |
| `Ctrl+q l` | 右のペインへ移動 |
| `Ctrl+Shift+[` | ペイン選択モード（番号で選択） |

**覚え方**: Vimと同じ h/j/k/l で移動

### ペインのサイズ調整

**リサイズモード**: `Ctrl+q s` でリサイズモードに入る

リサイズモード中:

| キー | 説明 |
|------|------|
| `h` | 左に拡大 |
| `j` | 下に拡大 |
| `k` | 上に拡大 |
| `l` | 右に拡大 |
| `Enter` | リサイズモードを終了 |

### ペインの最大化

| キー | 説明 |
|------|------|
| `Ctrl+q z` | 現在のペインを最大化/元に戻す（zoom） |

## コピーモード

ターミナルの出力をスクロールして、テキストをコピーできます。

### 基本操作

| キー | 説明 |
|------|------|
| `Ctrl+q [` | コピーモードに入る |
| `q` / `Escape` / `Ctrl+c` | コピーモードを終了 |
| `Enter` | 選択範囲をコピーして終了 |

### カーソル移動

| キー | 説明 |
|------|------|
| `h` / `j` / `k` / `l` | 左/下/上/右に移動 |
| `w` / `b` / `e` | 次の単語/前の単語/単語の末尾へ |
| `0` / `^` / `$` | 行頭/行頭（空白除く）/行末へ |
| `G` / `g` | 一番下/一番上へ |
| `H` / `M` / `L` | 画面上部/中央/下部へ |

### ジャンプ機能

| キー | 説明 |
|------|------|
| `f{文字}` | 次の{文字}にジャンプ |
| `t{文字}` | 次の{文字}の前にジャンプ |
| `F{文字}` | 前の{文字}にジャンプ |
| `T{文字}` | 前の{文字}の後にジャンプ |
| `;` | 前回のジャンプを繰り返し |

### スクロール

| キー | 説明 |
|------|------|
| `Ctrl+f` / `Ctrl+b` | 1ページ下/上へ |
| `Ctrl+d` / `Ctrl+u` | 半ページ下/上へ |

### テキスト選択

| キー | 説明 |
|------|------|
| `v` | 文字単位の選択モード |
| `V` | 行単位の選択モード |
| `Ctrl+v` | ブロック選択モード |
| `y` | 選択範囲をクリップボードにコピー |
| `o` / `O` | 選択範囲の始点と終点を入れ替え |

**使い方の流れ**:
1. `Ctrl+q [` でコピーモードに入る
2. `h/j/k/l` でカーソルを移動
3. `v` で選択開始
4. `h/j/k/l` で選択範囲を拡大
5. `y` でコピー、または `Enter` でコピーして終了

## その他の操作

### コピー＆ペースト

| キー | 説明 |
|------|------|
| `Cmd+c` | 選択範囲をコピー |
| `Cmd+v` | クリップボードから貼り付け |

### フォントサイズ

| キー | 説明 |
|------|------|
| `Ctrl++` | フォントサイズを拡大 |
| `Ctrl+-` | フォントサイズを縮小 |
| `Ctrl+0` | フォントサイズをリセット |

### コマンドパレット

| キー | 説明 |
|------|------|
| `Cmd+p` | コマンドパレットを表示 |
| `Ctrl+Shift+p` | コマンドパレットを表示 |

コマンドパレットでは、すべての操作を検索して実行できます。

### その他

| キー | 説明 |
|------|------|
| `Alt+Enter` | フルスクリーン切り替え |
| `Ctrl+Shift+r` | 設定ファイルを再読み込み |

## よく使う操作 Top 10

1. **`Ctrl+q d`** - 画面を縦に分割
2. **`Ctrl+q r`** - 画面を横に分割
3. **`Ctrl+q h/j/k/l`** - ペイン間を移動
4. **`Ctrl+q z`** - ペインを最大化/戻す
5. **`Cmd+t`** - 新しいタブを作成
6. **`Ctrl+Tab`** - 次のタブへ移動
7. **`Ctrl+q [`** - コピーモードに入る
8. **`Ctrl+q w`** - ワークスペース切り替え
9. **`Cmd+p`** - コマンドパレット表示
10. **`Ctrl+ +/-`** - フォントサイズ調整

## 設定ファイルの場所

- メイン設定: `~/.config/wezterm/wezterm.lua`
- キーバインド: `~/.config/wezterm/keybinds.lua`

設定を変更したら、`Ctrl+Shift+r` で再読み込みしてください。

---

## ヒント

- **Leader キーを忘れた場合**: `Ctrl+q` を押すと、2秒間Leaderモードになります
- **操作に迷った場合**: `Cmd+p` でコマンドパレットを開いて検索
- **コピーモードはVimライク**: Vimの操作を知っていれば直感的に使えます
- **ペイン移動もVimライク**: `h/j/k/l` でペイン間を移動できます
